# Deep Learning Models for Predicting Groundwater Levels in Data-Scarce Regions

This repository contains the datasets and source codes used in the study:

"Deep Learning Models for Predicting Groundwater Levels in Data-Scarce Regions: A Comparative Study of LSTM and CNN"

## Contents

- Raw groundwater level data
- Meteorological datasets
- CNN model implementation
- LSTM model implementation
- Reproducible training scripts
- Evaluation metrics

## Requirements

Python 3.11.5
